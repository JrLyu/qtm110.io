Probability Overview

Probability: the probability of an event AAA is the chance that AAA occurs. We denote it as P(A)P(A)P(A).

  • P(A)∈[0,1]=∈[0%,100%]P(A)\in[0,1]=\in[0\%,100\%]P(A)∈[0,1]=∈[0%,100%]

Probability Space: a probability space is a set of all possible outcomes of an experiment. We denote it as SSS.

  • Sample space SSS = total area. So, P(S)=1=100%P(S)=1=100\%P(S)=1=100%
  • We use Venn Diagram to depict probabilities graphically.

Independent Events: two events AAA and BBB are independent if they are unrelated. Independent events satisfy the following formula for their joint probability: P(A∩B)=P(A)⋅P(B). P(A\cap B)=P(A)\cdot P(B). P(A∩B)=P(A)⋅P(B).

Conditional Probability: the probability of an event AAA given that another event BBB has occurred. We denote it as P(A∣B)P(A\mid B)P(A∣B).

  • To calculate conditional probability, we have the following formula: P(A∣B)=P(A∩B)P(B). P(A\mid B)=\dfrac{P(A\cap B)}{P(B)}. P(A∣B)=​P(B)​​P(A∩B)​​.
  • The notation $\mid$ is called a pipe. It is read as "given that" or "conditional on".

Joint Probability: the probability of two events AAA and BBB occurring together. We denote it as P(A∩B)P(A\cap B)P(A∩B).

  • To calculate joint probability, we have the following formula: Joint Probability=P(A∩B)=P(A∣B)⋅P(B). \text{Joint Probability}=P(A\cap B)=P(A\mid B)\cdot P(B). Joint Probability=P(A∩B)=P(A∣B)⋅P(B).
  • Law of Total Probability: If we have an event BBB that takes on nnn values and an event AAA conditional on it, then P(a)=∑nP(A∣Bn)⋅P(Bn). P(a)=\sum_{n}P(A\mid B_n)\cdot P(B_n). P(a)=​n​∑​​P(A∣B​n​​)⋅P(B​n​​).
    • This is a simple-weighted average.
